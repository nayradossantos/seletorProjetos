Conteúdos:



HTML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar o conteúdo da web. Ele oferece novas funcionalidades e elementos semânticos para aprimorar a experiência do usuário e a eficiência do desenvolvimento web.

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais. O CSS trabalha em conjunto com o HTML, que define a estrutura do conteúdo, enquanto o CSS adiciona estilo e apresentação.

O que é a propriedade Display em CSS?


A propriedade .display em CSS define como um elemento HTML é exibido na página, determinando se ele se comporta como um elemento de bloco (ocupando toda a largura disponível) ou inline (ocupando apenas o espaço necessário para seu conteúdo e permitindo que outros elementos inline apareçam na mesma linha), além de definir o tipo de layout usado para seus filhos, como flexbox ou grid.

Display Características
Block A propriedade CSS display: block transforma um elemento em um bloco, fazendo com que ele ocupe toda a largura disponível em uma nova linha, empurrando os elementos seguintes para baixo. Elementos de bloco aceitam propriedades como width (largura), height (altura), margin (margem) e padding (preenchimento), oferecendo mais flexibilidade na estilização do que elementos inline.
Inline Um elemento com display: inline é exibido na mesma linha que outros elementos, ocupando apenas o espaço necessário para seu conteúdo (o que é diferente de block, que vai ocupar todo o espaço disponível, mesmo que sobre espaço após o seu conteúdo).
Inline-Block Em CSS, display: inline-block combina características dos layouts inline e block. Elementos com essa propriedade se comportam como elementos inline, permitindo que vários elementos fiquem na mesma linha. No entanto, ao contrário de inline, eles também aceitam a definição de width e height, assim como elementos block.
Flex É uma propriedade CSS que transforma um elemento HTML em um container flexível, permitindo que seus filhos (elementos dentro dele) sejam organizados e alinhados de forma flexível em uma ou duas dimensões. É uma ferramenta poderosa para criar layouts responsivos e adaptáveis a diferentes tamanhos de tela.
